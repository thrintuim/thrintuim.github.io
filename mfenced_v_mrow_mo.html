menclose vs mrow mo fence announcement

The following is to demonstrate how iOS VoiceOver does and does not handle parentheses in text and MathML

Parens in text (this is a parenthetical statement).

mrow mo markup: (1+1)

mfenced markup: 1+1

Set notation mrow mo markup: {x|−3<x≤9}

Set notation with with mfenced markup: x|−3<x≤9

Absolute value equation with mrow mo markup: |−2−|3||

Absolute value equation with mfenced markup: −2−3

function notation mrow mo: f(x)

function notation mfenced: fx
