Types of Real Numbers

The Natural (or Counting) Numbers: This is the set of numbers N = 1 2 3 4 5 … . The set is infinite, so in list form we can write only the first few numbers.

The Whole Numbers: This is the set of natural numbers and 0 : 0 1 2 3 4 5 … . Again, we can list only the first few members of this set. No special symbol will be assigned to this set in this course.

The Integers: This is the set of natural numbers, their negatives, and 0 . As a list, this is the set Z = … − 4 − 3 − 2 − 1 0 1 2 3 4 … . Note that the list continues indefinitely in both directions.

The Rational Numbers: This is the set, with symbol Q (for quotient), of ratios of integers (hence the name). That is, any rational number can be written in the form p q , where p and q are both integers and q ≠ 0 . When written in decimal form, rational numbers either terminate or have a repeating pattern of digits past some point.

The Irrational Numbers: Every real number that is not rational is, by definition, irrational. In decimal form, irrational numbers are nonterminating and nonrepeating. No special symbol will be assigned to this set in this course.

The Real Numbers: Every set above is a subset of the set of real numbers, which is denoted R . Every real number is either rational or irrational, and no real number is both.
